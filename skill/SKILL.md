一、技能概述
技能名称
AI错题教练
适用人群
中小学生、课外辅导学习者，存在错题整理繁琐、复习无针对性、薄弱知识点难以定位的学习痛点。
核心定位
依托大语言模型打造的智能错题管理工具，区别于传统纸质错题本、普通表格记录工具，依靠AI实现自动错因分析、知识点匹配、变式习题生成、个性化复习规划，一站式完成错题收纳与针对性查漏补缺。
二、核心功能
1. 错题信息录入
支持手动录入题干、学生错误答案、标准答案，区分数学、英语、物理等中小学主流学科，自动标记对应年级。
2 AI错因智能诊断
自动划分四类错误根源：计算失误、概念混淆、审题不清、解题思路缺失，精准定位学生思维漏洞。
3. 知识点自动匹配
对接国内中小学标准知识点库，给每道错题绑定对应考点，汇总个人高频薄弱知识点。
4. 同考点变式题生成
针对单道错题自动生成2-3道同等难度、同一考点的全新练习题，杜绝重复刷题，强化薄弱点。
5. 自适应复习规划
结合艾宾浩斯遗忘曲线、个人错题错误程度，动态生成分阶段复习清单，规划每日复习任务。
6. 阶段性学习报告导出
汇总周期内全部错题，统计错误类型占比、薄弱知识点排名，输出文字版学习提升建议，可导出保存。
三、运行环境与依赖
1. 基础环境：Python 3.9 及以上版本
2. 第三方库：pandas（表格存储）、openai（大模型调用）、python-dotenv（配置读取）
3. 外部依赖：大语言模型API密钥，需配置.env文件填写密钥信息
四、目录结构说明
skill/
├── SKILL.md                # 本技能说明文档
├── scripts/
│   └── error_coach.py      # AI错题教练主运行程序
└── references/
    ├── .env.example       # API密钥配置模板
    └── requirements.txt    # 项目依赖库清单
五、完整使用步骤
1. 环境配置
打开references/.env.example，复制生成.env文件，填入有效的大模型API密钥；在终端执行pip install -r references/requirements.txt安装全部依赖包。
2. 启动程序
终端进入skill/scripts目录，运行命令：python error_coach.py启动工具。
3. 功能选择
程序启动后显示功能菜单，按需选择：录入错题/查看错题库/生成变式习题/生成复习计划/导出学习报告。
4. 数据保存
所有错题数据自动存储至Excel表格，生成的报告可本地导出留存。
六、AI核心价值说明
1. 传统工具仅能单纯记录错题，无法自动分析错误思维，本技能依靠大模型自动拆解错因，人工批量分析效率极低；
2. 普通题库只能固定出题，AI可根据个人专属错题实时定制变式题，精准匹配个人薄弱点；
3. 人工规划复习耗时费力，AI结合遗忘曲线动态调整复习周期，实现千人千面的个性化学习方案。
七、迭代优化预留方向
1. 多模态OCR拍照录入错题，支持直接拍摄试卷自动识别题干与答案；
2. 学习报告可视化，生成柱状图、饼图直观展示错题分布；
3. 知识点掌握度分级，根据错题正确率调整复习任务轻重。
